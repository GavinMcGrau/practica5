# Catalogo películas (Cliente)

## Cómo instalar el proyecto
```
npm install
```

### Compila y realiza reinicios rapidos para el desarrollo
```
npm run serve
```

### Compila y deja el proyecto menos pesado para producción
```
npm run build
```

### Comprueba la sintaxis y arregla fallos
```
npm run lint
```
